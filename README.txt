# My CV Website

A personal CV/portfolio website built using HTML and CSS. This website showcases my background, education, skills, projects, and contact information in a clean and simple layout.

## Features

* Responsive webpage structure
* Navigation menu with section links
* About Me section
* Education details
* Skills list
* Project showcase
* Contact information
* Footer section

## Technologies Used

* HTML5
* CSS3

## Project Structure

```bash
index.html
style.css
README.md
```

## Sections Included

### About Me

A short introduction about myself, my passion for web development, and my personal strengths.

### Education

Information about my educational background.

### Skills

A list of personal and professional skills.

### Projects

Details about projects and learning experiences.

### Contact

My email address and phone number.

## How to Run the Project

1. Download or clone the project files.
2. Make sure `index.html` and `style.css` are in the same folder.
3. Open `index.html` in any web browser.
